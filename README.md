# mctx Claude Code Plugins

Auto-generated Claude Code plugin marketplace for [mctx](https://mctx.ai)-hosted MCP servers.

## For subscribers

Add the mctx marketplace to Claude Code:

```
/plugin marketplace add https://api.mctx.ai/marketplace.json
```

Then install any server:

```
/plugin install {slug}@mctx
```

## For developers

Enable "Claude Code" publishing in your [server dashboard](https://app.mctx.ai/dev/servers) to automatically publish your MCP server as a Claude Code plugin.
